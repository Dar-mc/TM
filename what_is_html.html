HTML

HTML est l'abréviation de HyperText Markup Language. C'est le langage de base utilisé pour créer et structurer du contenu web. Il permet de placer les éléments dans la page pour qu'ils soient affichés correctement.

Chaque élément est défini entre des balises (<nom de l'élément>) et doit être refermée avec </nom de l'élément>. Chaque page HTML doit être ouverte par <!DOCTYPE html>, puis <html> et fermée par </html>.

La plupart des pages ont un en-tête HTML au début (à ne pas confondre avec l'en-tête (header) présenté plus tard). On y met le titre, les métadonnées, les liens, etc...

Le titre de la page est le nom qui apparaît dans l'onglet du navigateur. Il est défini par <title>.

Les métadonnées sont les informations visibles qui décrivent le contenu ou les paramètres de la page (ex:encodage, description, mots-clés).

Les liens sont utilisés pour lier la page avec une source externe, trouvable sur internet ou dans le dossier de la page. Ils sont souvent utilisés pour mettre des liens aux fiches de style CSS, qui permettent de modifier le style de la page et définissent son apparence visuelle. Ils sont utilisés avec le format <link rel="type" href="lien"<. La relation (rel="") est le la nature du lien entre la page et la source externe. Il y a beaucoup de natures différentes, comme les fiches de style (stylesheet), les icones (icon) ou les préchargement de ressources (preload). La référence hypertexte (href="") est l'endroit où l'on indique l'adresse du chemin vers la source que l'on veut lier.

Le corps suit directement l'en tête. Il est désigné par <body>. Il contient tout le contenu du site que l'utilisateur peut voir.

Pour ajouter du texte sur une page web, vous pouvez utiliser les titres. Ils permettent de hiérarchiser les textes avec différentes tailles, <h1> étant le plus grand et <h6> étant le plus petit. Il est aussi possible d'utiliser des paragraphes <p>. Ces derniers n'ont qu'une taille.

Il est possible d'ajouter des images sur une page web. Pour cela, utilisez <img src="lien/chemin de l'image">.

Les conteneurs sont des éléments qui peuvent être remplis avec d'autres éléments, comme du texte ou des images. Ils sont utiles pour organiser la structure du site. Utilisez-les avec <div>. Ils sont par défaut des blocks transparents et rectangulaires. Il est possible de les rendre différents en définissant le style dans la balise ou avec des propriétés CSS. Le corps et la tête de pages sont des sortes de conteneurs.

Il est aussi possible de faire des formulaires, tableaux, listes, sauts de ligne, d'ajouter des vidéos, d'intégrer des pages internet et bien plus. Si vous voulez en apprendre plus, nous vous recommendons le site W3Schools qui offre des cours plus approfondis et couvre un plus large panel d'éléments. Leurs cours sont très accessibles aux débutants et sont également utiles pour les personnes plus expérimentées. Cette page parlait principalement des grandes bases d'HTML et des éléments utilisés sur notre site.

HTML ne permet ni de changer le style pour tous les éléments de la page (polices, couleurs, etc...), ni de faire des intéractions dynamiques (animations ou autre). Pour le style vous devez utiliser CSS.
